# Portfolio Website

A personal portfolio website showcasing my profile, skills, and selected projects.  
Built with a focus on responsiveness, performance, and clean UI behavior across all devices.

---

## Highlights

- Responsive design using CSS media queries  
- Lazy loading for images to improve performance  
- Light / Dark mode with persistence using `localStorage`  
- Cross-platform support (mobile, tablet, desktop)

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## Tooling & Deployment

- **Vite** – Development and build tool  
- **Vercel** – Hosting and deployment
